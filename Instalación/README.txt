Tutorial de ubicación en los binarios del Sistema y ejecución del Juego:


Sobre la consola del Sistema: 
>Escribir 'bash Makefile' para generar el ejecutable del juego 
(Para esta compilación se necesitan las bibliotecas Pthread y Ncurses de C).

>Escribir 'bash Install' para ubicar el ejecutable en los binarios del Sistema 
(así puede abrirse desde cualquier dirección en la consola).

>Escribir en la consola 'MatComInvasion' para ejecutar el juego.

**Para eliminar el ejecutable de los binarios del sistema escriba en la consola 'sudo rm /usr/bin/MatComInvasion'. 
En caso de error puede consultar con el comando 'which MatComInvasion' que devolverá la ubicación del ejecutable, 
y para eliminar bastaría 'sudo rm <ubicación>'.
